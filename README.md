## 如何注册Patreon账户

1. 访问 [Patreon注册页面](https://www.patreon.com/login)。
2. 输入你的邮箱地址，设置昵称和密码，完成基本信息填写。
3. 验证邮箱，确保你能正常接收来自 Patreon 的通知。


## 如何在Patreon上订阅

1. 在 Patreon 主页面中，点击 “查找创作者”。
2. 输入创作者名称，例如 `Best Kept Secret`，并选择对应的创作者页面。
3. 选择适合你的订阅档位，并点击加入。
